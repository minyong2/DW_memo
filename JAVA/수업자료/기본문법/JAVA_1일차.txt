### 프로그래밍 
    모든 프로그래밍 언어는 위에서 아래로 해석하며,
    오른쪽에서 왼쪽으로 계산 및 대입한다.

### 자바란?
    *객체(Object)지향 프로그래밍 언어
    *요새 나오는 언어들은 객체지향
    *협업과 코드를 관리하기 위해서는 객체지향이 필요 

### 프로젝트 -> 패키지 -> 클래스 파일
    프로젝트 안에 여러 패키지가 존재 패키지 안에 클래스 파일이 존재
    클래스 파일은 우리가 코딩할 코드 파일
    ex) 데이터베이스 -> 테이블 -> 데이터

### 클래스와 메인 함수
    클래스는 함수들을 담는 그릇
    함수들은 코드들을 담는 그릇

### 변수란
    데이터를 저장하는 공간이며 사용 및 선언 방법이 C/C++와 동일.
    ex) int num = 10;  // num이 변수

### 데이터 타입이란
    숫자로 데이터를 저장할지 문자로 데이터를 저장할지 결정.
    자바에는 총 8가지 기본 데이터 타입과 1가지 클래스 타입이 존재.
    즉, 자바는 총 9가지 데이터 타입을 가지고 있음.
    ex) 정수형 데이터타입 int, 실수형 데이터타입 double, 문자형 데이터타입 String

### 변수와 데이터 타입 규칙
    하나 이상의 글자로 이루어져야 함
    길이 제한이 없음
    '$', '_' 이외의 특수문자 사용 불가능
    두번재 이후의 글자는 숫자, 문자, '$', '_'이어야 함
    변수이름은 중복 불가능(같은 지역변수에 있다면)

### JAVA 기본 연산자
    증감 연산자 : ++, -- 
        ex) 1씩 증가 혹은 감소 하겠다.
    사칙 연산자 :  +, -, *, / , %(나머지 값)
    삼항 연산자 : (조건식) ? A(TRUE) : B(FALSE)
        ex) int num = (4 > 2) ? 3 : 5;
    비교 연산자 : ==, !=, >, <,>=, <=
    문자열 연산자 : +
    대입 연산자 : =, *=, /=, %=, +=, -=, !=
        ex) 프로그래밍에서는 =는 같다가 아니라 대입을 의미.
        ex) int sum = sum + sum은 int sum += sum 으로 가능.
