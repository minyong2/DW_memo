아래 기본 명령어만 숙지하자.

### 자주 사용하는 EXCEL 단축키

1. 셀 병합 단축키 및 커스텀
    => 셀 병합 : alt + h + m + c
    => 단축키는 있지만 단축키 쓰는 것이 클릭하는 것보다 어려운 기능들이 존재, 
    엑셀에는 수많은 기능들이 있기 때문에.
    단축키를 새롭게 만들어야 함( = 커스텀 해야한다.)

    파일 -> 옵션 -> 빠른 실행 도구 모음 -> 병합하고 가운데 맞춤 선택 후 추가 -> alt + 4 세팅 완료

2. 셀 높이, 길이 조절 단축키 : alt + h + o + h 
    마우스로 하는걸 권장

3. 뒤로 가기 단축키 : ctrl + z

4. 엑셀 줄바꿈 : alt + enter

5. 찾기 : ctrl + F

6. 잘라내기 : ctrl + x -> ctrl + v

7. 모두 선택 / 선택하고 싶은 것 : ctrl + a / ctrl + 마우스 왼쪽 클릭 

8. 정렬 버튼 만들기
    해당 셀 클릭 -> 데이터 -> 필터

9. 함수 생성 단축 키 : shift + f3
    sum을 많이 사용

10. 틀 고정 : 보기 -> 틀 고정 (원하는 행이 있다면, 다음 행을 선택하여 고정)
