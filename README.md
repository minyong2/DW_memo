## 수업 소개
***

## 수업 목표 ⭐
***
-   IT기본 용어와 컴퓨터 기초 능력(문서 능력, 단축키, 기타 Tip)을 키우자.

-   기본 SQL 문법을 익히고 DB 테이블 설계를 해보자.
-   Spring을 이용하여 웹을 개발하자.
-   Android의 원리와 기본 동작을 배워보자.
-   HTML, CSS, Javascript을 이용하여 웹을 디자인 해보자.
-   코드리뷰을 통해 실무능력을 키우자.
